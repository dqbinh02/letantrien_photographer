# Photographer

A minimal photographer website with album management and gallery display.

## Getting Started

Install dependencies:

```bash
pnpm install
# or
npm install
```

Run the development server:

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Pages

- **Album**: Display your photo albums (renamed from blog)
- **Gallery**: Photo gallery view

## Built With

- Next.js 15
- React 19
- Once UI System
- TypeScript
- Tailwind CSS
